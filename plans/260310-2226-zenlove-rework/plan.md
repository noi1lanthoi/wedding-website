# Plan: Zenlove Rework

Created: 2026-03-10

## Overview

Đập đi xây lại giao diện trang web cưới để giống với mẫu zenlove.me (Mẫu thiep-cuoi-20-pre). Giao diện chủ đạo là Mobile-First, nền giấy scrapbook, kết hợp mảng xanh rêu và font chữ nghệ thuật sang trọng.

## Tech Stack

- Frontend: Next.js + Tailwind CSS
- Animations: Framer Motion
- Icons: Lucide React
